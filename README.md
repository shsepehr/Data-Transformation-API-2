# FastAPI Salary per Age API

This project provides a simple **FastAPI** endpoint that receives `age` and `salary` as input and returns a calculated metric (**salary_per_age**) as output.  
It demonstrates API development, data validation, and lightweight computation using FastAPI + Pydantic.
